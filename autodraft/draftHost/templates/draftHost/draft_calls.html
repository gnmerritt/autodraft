All the following require a valid key parameter. Click on the links for an example response.

What URL pattern Returns
Draft info: {% url 'draftHost:api:draft' %} Draft information, teams, picks
Your team: {% url 'draftHost:api:current_team' %} { name:"Your name", picks: [list of pick objects], players: [who you've drafted] }
Draft picks & selections: {% url 'draftHost:api:picks' %} picks & selections
Make a pick: {% url 'draftHost:api:make_pick' 1623 %} { code: pick status code, success: True or missing, message: human readable state, selection: fantasy selection object }
Player status in your draft: {% url 'draftHost:api:player_status' 1623 %} Returns the player object, including fantasy team if he's been picked. Use this if you want to check player availability before your turn!
