What URL pattern Returns
Player by id: {% url 'draftHost:player' 1 %} returns a player object
Player search:/search/{query}returns a list of matching players. For best results query should be the last name of a player
Team by id:/team/{id} or /team/{name}returns info about another fantasy team
All NFL Teams: {% url 'draftHost:nfl_teams' %} {teams: [list of teams]}
Players on a specific NFL team:/nfl/team/{id}/players/NFL Team object with extra field "players"
Players on a fantasy team:/team/{id}/playersReturns a list of player objects
NFL Conferences: {% url 'draftHost:nfl_conferences' %} {divisions: [list of divisions]}
NFL Divisions: {% url 'draftHost:nfl_divisions' %} {conferences: [list of conferences]}
Colleges {% url 'draftHost:colleges' %} {colleges: [list of colleges]}
Players by College {% url 'draftHost:college_players' 1 %} {college object with players list}
